# Product Overview

## Product Name
**CVTailor** — AI-Powered CV Tailoring & Interview Prep Platform

## Tagline
*Paste a job. Get a tailored CV, interview plan, and mock interview — in minutes.*

## What It Is
CVTailor is a web application that takes a job listing (title, description, requirements) as input and delivers three intelligent outputs:

1. **Tailored CV** — Rewrites and reprioritises the user's uploaded CV to match the target job's keywords, tone, and requirements.
2. **Interview Prep Plan** — A structured, role-specific preparation roadmap covering key topics, likely question categories, company research tips, and a suggested study schedule.
3. **Mock Interview** — An interactive Q&A session where an AI interviewer asks role-relevant questions, the user answers, and receives instant feedback on content, structure, and delivery.

## Target Users
- Job seekers (fresh graduates, mid-career professionals, career switchers) who want to maximise their CV-to-interview conversion rate.
- Recruitment bootcamp participants and career coaches who want a tool to run mock sessions.

## Core User Flow
1. User uploads their existing CV (PDF or paste text).
2. User pastes or types the job entry (job title + description + requirements).
3. The app analyses the gap between the CV and the job.
4. Output A: Tailored CV preview + download.
5. Output B: Personalised Interview Prep Plan (structured document/checklist).
6. Output C: Interactive Mock Interview session with AI feedback per answer.

## Positioning
CVTailor sits at the intersection of ATS optimisation tools (Resume.io, Jobscan) and interview prep tools (Interviewing.io, Pramp) — but unifies them in a single job-entry-driven workflow. It is especially differentiated by the **job-entry-first** approach: everything flows from the specific role the user is targeting.

## Brand & Tone
- Professional yet approachable; confident and empowering.
- Language: primarily Bahasa Indonesia (with English toggle planned).
- Visual identity: clean, minimal, trust-inspiring — navy/white/accent-green palette.

## Scope (MVP)
- Job entry input (free-text paste or URL scrape).
- CV upload (PDF) + text extraction.
- AI-driven CV tailoring with diff/highlight view.
- Interview prep plan generation (downloadable PDF/checklist).
- Mock interview module: AI asks → user answers (text) → AI scores + feedback.
- User authentication (sign-up/login) and session persistence.

## Strategic Principles
- **Job-first**: every feature starts from the target job, not the CV.
- **Actionable output**: users leave with files and plans they can use immediately.
- **Feedback loop**: mock interview responses improve the prep plan iteratively.
- **Privacy**: CV data is not stored beyond the active session without explicit consent.
